# swu-grade-calculator-web

西南大学成绩单均分与绩点计算器。项目使用 Vite + React + TypeScript 开发，所有成绩单解析和计算都在浏览器本地完成，不会把用户上传的 `.xlsx` 文件发送到服务器。

## 功能

- 上传 `.xlsx` 成绩单并读取第一个工作表。
- 自动模糊识别课程名称、学分、成绩列，并支持手动选择列。
- 清洗缺失或无效数据，展示被忽略数据。
- 同一课程重复出现时，保留成绩最高的一条；成绩相同时保留学分较高的一条。
- 计算算数平均分、加权平均分、平均学分绩点 GPA / 5.00。
- 支持手动新增课程 / 模拟成绩，并可删除模拟课程。
- 支持目标算数平均分、目标加权平均分分析。
- 导出包含汇总结果、去重后课程表、重复课程对比表、被忽略数据的 `.xlsx` 文件。

## 安装

```bash
npm install
```

## 本地运行

```bash
npm run dev
```

运行后在浏览器打开终端中显示的本地地址，通常是 `http://localhost:5173/`。

## 打包

```bash
npm run build
```

打包产物会生成在 `dist/` 目录。

## 预览打包结果

```bash
npm run preview
```

## 使用说明

1. 点击页面顶部上传区域，选择西南大学成绩单 `.xlsx` 文件。
2. 系统会自动读取第一个工作表并识别列名。
3. 如果课程名称、学分、成绩列没有被正确识别，请在“列识别”区域手动选择。
4. 查看三个结果卡片、详细统计、去重后课程表、重复课程对比表和被忽略数据。
5. 可以在“手动新增课程 / 模拟成绩”中输入课程名称、学分、成绩，模拟还未出的课程成绩。
6. 可以输入目标均分查看距离目标还差多少。
7. 点击“导出结果”下载计算后的 `grade-calculation-result.xlsx`。

## 计算规则

算数平均分：

```text
Σ成绩 / 课程门数
```

加权平均分：

```text
Σ(成绩 × 学分) / Σ学分
```

平均学分绩点：

```text
Σ(课程学分 × 相应绩点) / Σ课程学分
```

五级制成绩会先转换成百分制：

- 优秀 / A = 95
- 良好 / B = 85
- 中等 / C = 75
- 及格 / D = 65
- 不及格 / E = 55

项目不会直接使用成绩单中的绩点列，而是始终根据百分制成绩自动换算西南大学 5 分制绩点。

## 隐私说明

本项目没有后端服务，文件解析、数据清洗、重复课程处理、计算和导出都在浏览器本地完成。请不要把真实成绩单文件提交到 GitHub；`.gitignore` 已默认忽略常见表格文件和构建产物。
